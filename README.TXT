Overzicht:
  Export-plugin-juizs.php: 
    Deze wordpress plugin had ik helemaal op maat gemaakt voor de klant juizs (www.juizs.nl).
    De reden dat we niet een normale woocommerce export plugin gebruikt kon worden is omdat de pakketten van juizs anders werken dan de normale producten in woocommerce.
    Elke product (pakket) bevat meerdere flesjes. De leverancier moest een overzicht hebben in Excel waar alle flesjes los als rij worden weergegeven.  Dit is de reden dat ik ben begonnen aan een eigen gemaakte export plugin voor juizs. 
    De code is een beetje omslachtig omdat ik weinig tijd had gekregen, hierdoor moest ik snel werken en had ik geen tijd om deze code te reviewen. Achteraf gezien vind ik dit jammer omdat ik hier heel veel verbeter punten in zie.
  Front-controller.php:

  Team-slider.js:
    Dit is de een van de sliders op madpack.works. Deze slider zorgt er voor dat je kan slepen, hoveren en klikken op de teamfoto van madpack.
    Wanneer je sleept gaat de foto van links naar rechts,
    Wanneer je hovert of klikt (mobiel) word de persoon uitgelicht.
    Als je niet met je muis boven of op de teamfoto hangt gaat de foto van links naar recht animeren.
    Bekijk het online: www.madpack.works
